A programming language created in Python


LION is an interpeted language, written in Python. Its commands are:

### I/O

`write`
`roar`
`input`

### Control Flow

`if`
`for`
`while`


### Functions

`function`
`call`
`return`

### Typecasting

`int`
`string`
`float`

### Variables

`var`
`savedVar`
`savedVarDelete`
`array`

### Other

`system`
